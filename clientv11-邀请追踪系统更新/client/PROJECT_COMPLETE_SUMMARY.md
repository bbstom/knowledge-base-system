# 信息搜索平台 - 完整项目总结

## 📋 项目概述

这是一个功能完整的信息搜索平台管理系统，包含前台用户界面和后台管理系统。系统支持多种搜索类型、积分系统、佣金系统、推荐奖励、USDT提现等完整功能。

**项目名称：** InfoSearch Platform  
**版本：** 1.0.0  
**最后更新：** 2024-10-19  
**技术栈：** React 19 + TypeScript + Tailwind CSS v4 + Vite

## 📊 数据流转

### 用户注册流程
```
用户注册 → 获取注册奖励积分 → 如有推荐码 → 推荐人获得奖励 → 被推荐人获得奖励
```

### 搜索流程
```
选择搜索类型 → 输入搜索内容 → 检查积分余额 → 扣除积分 → 执行搜索 → 返回结果 → 记录历史
```

### 推荐流程
```
用户A生成推荐链接 → 用户B通过链接注册 → 系统记录推荐关系 → 发放推荐奖励积分
```

### 佣金流程
```
用户B充值 → 系统计算佣金 → 发放给推荐人A → 记录佣金记录 → 用户A可提现
```

### 提现流程
```
用户申请提现 → 填写USDT地址 → 提交申请 → 管理员审核 → 批准/拒绝 → 发送USDT → 标记完成
```

## 💰 佣金系统说明

### 佣金计算

#### 一级佣金
- 直接推荐的用户充值
- 默认比例：10%
- 即时到账

#### 二级佣金
- 推荐用户的推荐用户充值
- 默认比例：5%
- 即时到账

#### 三级佣金
- 二级推荐用户的推荐用户充值
- 默认比例：2%
- 即时到账

### 结算方式

**即时结算：**
- 用户充值后立即发放佣金
- 适合小额佣金
- 用户体验好

**每日结算：**
- 每天固定时间统一结算
- 适合中等规模平台
- 便于管理

**每周结算：**
- 每周固定时间统一结算
- 适合大规模平台
- 降低系统负担

**每月结算：**
- 每月固定时间统一结算
- 适合企业级平台
- 便于财务管理

### 提现规则

**最低提现金额：**
- 默认：100元
- 可在后台配置

**提现手续费：**
- 默认：5%
- 可在后台配置

**USDT汇率：**
- 实时汇率或固定汇率
- 可在后台配置

**审核方式：**
- 自动审核：小额提现自动通过
- 人工审核：大额提现需要人工审核
- 自动审核上限可配置

## 🚨 常见问题

### 开发相关

**Q: 如何启动项目？**
A: 运行 `npm install` 安装依赖，然后运行 `npm run dev` 启动开发服务器。

**Q: 如何构建生产版本？**
A: 运行 `npm run build` 构建，然后运行 `npm run preview` 预览。

**Q: 如何修改端口？**
A: 在 `vite.config.ts` 中修改 `server.port` 配置。

### 功能相关

**Q: 如何添加新的搜索类型？**
A: 访问管理后台 → 系统设置 → 搜索类型 → 添加。

**Q: 如何修改积分规则？**
A: 访问管理后台 → 系统设置 → 积分配置。

**Q: 如何处理提现申请？**
A: 访问管理后台 → 提现管理 → 查看详情 → 批准/拒绝。

### 技术相关

**Q: 当前使用的是真实数据还是模拟数据？**
A: 当前使用模拟数据（mockDatabase.ts），生产环境需要连接真实后端API。

**Q: 如何连接真实后端？**
A: 修改 `src/utils/api.ts` 中的 API 地址，实现真实的API调用。

**Q: 如何部署到生产环境？**
A: 构建项目后，将 `dist` 目录部署到Web服务器（如Nginx、Apache）。
