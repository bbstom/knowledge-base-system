## 🏗️ 技术架构

### 前端技术栈
```
React 19 + TypeScript + Tailwind CSS v4
├── 路由管理: React Router v6
├── 状态管理: React Hooks + Context
├── 表单处理: React Hook Form + Zod
├── HTTP 客户端: Axios
├── 图标库: Lucide React
├── 通知组件: React Hot Toast
├── 工具库: clsx, tailwind-merge
└── 构建工具: Vite
```

### 后端架构（待实现）
```
Node.js + Express + MongoDB
├── 用户数据库: MongoDB (独立服务器)
├── 查询数据库: 独立服务器部署
├── 支付网关: BEpusdt (USDT)
├── 邮件服务: SMTP
├── 认证系统: JWT
└── 缓存系统: Redis
```

## 🎯 核心业务流程

### 用户注册流程
1. ✅ 邮箱注册 + 推荐码（可选）
2. 🔄 邮箱验证链接发送
3. 🔄 邮箱验证完成
4. ✅ 自动登录到仪表盘

### 搜索流程
1. ✅ 选择搜索类型和数据库
2. ✅ 输入查询内容
3. ✅ 显示费用预览
4. 🔄 余额扣费
5. 🔄 执行查询
6. ✅ 返回结果展示

### 推荐奖励流程
1. ✅ 生成推荐码和链接
2. ✅ 好友通过推荐注册
3. 🔄 好友完成邮箱验证
4. 🔄 推荐人获得注册奖励
5. 🔄 好友消费时推荐人获得佣金

### 提现流程
1. ✅ 填写提现金额和 USDT 地址
2. ✅ 提交提现申请
3. 🔄 管理员审核
4. 🔄 USDT 转账处理
5. 🔄 提现状态更新
