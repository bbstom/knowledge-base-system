# 🚀 快速开始测试

## 一键启动指南

### Windows用户

**1. 启动后端（终端1）：**
```cmd
cd server
npm install
npm start
```

**2. 启动前端（终端2）：**
```cmd
npm run dev
```

### Mac/Linux用户

**1. 启动后端（终端1）：**
```bash
cd server
npm install
npm start
```

**2. 启动前端（终端2）：**
```bash
npm run dev
```

## 🐛 遇到问题？

### 后端启动失败
- 检查MongoDB是否运行
- 检查端口3001是否被占用
- 查看server/.env配置

### 前端启动失败
- 检查端口5173是否被占用
- 运行 `npm install` 安装依赖

### 注册失败
- 检查后端是否运行
- 查看浏览器Console错误
- 查看后端日志

### 签到失败
- 检查是否已登录
- 查看后端日志
- 检查数据库连接
