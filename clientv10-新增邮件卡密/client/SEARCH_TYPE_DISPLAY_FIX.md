# 搜索类型显示优化

## 问题描述

搜索历史记录中：
- 搜索类型显示为空或英文代码（如 "phone", "idcard"）
- 查询内容可能显示为 "N/A"

## 修复方案

### 文件：`src/pages/Dashboard/Orders.tsx`

**改进内容**:

1. **添加搜索类型中文映射**
```typescript
const typeMap: Record<string, string> = {
  'phone': '手机号',
  'idcard': '身份证',
  'name': '姓名',
  'email': '邮箱',
  'address': '地址',
  'company': '公司',
  'vehicle': '车辆',
  'other': '其他'
};
const typeLabel = typeMap[log.type] || log.type || '未知';
```

2. **添加空值处理**
```typescript
{log.query || 'N/A'}
{log.pointsCost || 0} 积分
{log.resultCount || 0} 条
```
