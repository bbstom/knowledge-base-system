## 🚀 快速配置

### 步骤1：配置BEpusdt
1. 登录管理员账号
2. 进入"网站配置"页面
3. 找到"充值系统配置"部分
4. 填写：
   - BEpusdt服务地址
   - API密钥
   - 商户ID
   - 选择支持的币种

### 步骤2：设置充值套餐
1. 在"充值套餐"部分
2. 点击"+ 添加套餐"
3. 设置积分数量和金额
4. 勾选"启用"
5. 点击"保存配置"

### 步骤3：测试充值
1. 访问 `/dashboard/recharge`
2. 选择充值套餐
3. 选择支付币种
4. 创建测试订单
