# 时区同步功能 - 快速指南

## 🎯 功能说明

在时区配置页面中，系统会自动检测您的本地时区，并提供一键同步功能。

## 📱 界面说明

### 蓝紫色卡片 - 服务器时间
显示当前服务器配置的时区和时间

### 绿色卡片 - 您的本地时间
- 自动检测您的浏览器时区
- 显示您的本地时间
- 提供"同步此时区"按钮

## ⚡ 快速使用

### 方法1：一键同步（推荐）
```
1. 打开时区配置页面
2. 查看绿色卡片中的本地时间
3. 点击"同步此时区"按钮
4. 点击"保存配置"
5. 重启服务器
```

### 方法2：手动选择
```
1. 在下拉列表中选择时区
2. 选择时间显示格式
3. 点击"保存配置"
4. 重启服务器
```

## 💡 使用场景

### 场景1：首次配置
新部署的系统，快速设置为本地时区

### 场景2：多地管理
管理员在不同地区访问，快速切换时区

### 场景3：时区迁移
服务器迁移到新地区，快速更新时区

## ✅ 优势

- 🚀 **快速** - 一键同步，无需查找
- 🎯 **准确** - 自动检测，避免错误
- 👁️ **直观** - 实时对比，清晰明了
- 💪 **智能** - 自动匹配最接近时区
