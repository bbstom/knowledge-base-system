## 📋 功能概述

全新的充值中心让用户可以清晰地选择：
- **积分充值** - 按需购买积分，用于搜索服务

## 🔧 管理员配置指南

### 步骤1：配置BEpusdt支付

1. 登录管理员账号
2. 进入"网站配置"页面
3. 找到"充值系统配置"部分
4. 填写以下信息：
   ```
   BEpusdt服务地址: https://your-bepusdt-domain.com
   API密钥: your-api-key-here
   商户ID: your-merchant-id-here
   支持币种: ☑ USDT  ☑ TRX
   ```
5. 点击"保存配置"

### 步骤2：配置积分充值套餐

1. 在"积分充值套餐"部分
2. 点击"+ 添加套餐"
3. 设置：
   - 积分数量：100
   - 金额（元）：10
   - ☑ 启用
4. 重复添加更多套餐
5. 点击"保存配置"

**推荐套餐配置：**
```
100积分 / 10元
500积分 / 50元
1000积分 / 100元
2000积分 / 200元
5000积分 / 500元
10000积分 / 1000元
```

### 步骤3：配置VIP会员套餐

1. 在"VIP会员套餐"部分
2. 点击"+ 添加VIP套餐"
3. 设置：
   - 套餐名称：月度VIP
   - 有效天数：30
   - 金额（元）：30
   - VIP特权（每行一个）：
     ```
     无限搜索次数
     专属客服
     优先数据更新
     去除广告
     ```
   - ☑ 启用
4. 重复添加更多套餐
5. 点击"保存配置"

**推荐VIP套餐配置：**

**月度VIP（30天/30元）**
```
无限搜索次数
专属客服
优先数据更新
去除广告
```

**季度VIP（90天/80元）** ⭐推荐
```
无限搜索次数
专属客服
优先数据更新
去除广告
赠送500积分
```

**年度VIP（365天/300元）**
```
无限搜索次数
专属客服
优先数据更新
去除广告
赠送2000积分
VIP专属标识
```

## 👤 用户使用指南

### 访问充值中心

**方式1：侧边栏导航**
- 点击左侧菜单"充值中心"

**方式2：Dashboard快捷入口**
- 在Dashboard页面点击"充值中心"卡片

**方式3：直接访问**
- 访问 `/dashboard/recharge-center`

### 充值积分流程

1. **进入充值中心**
   - 默认显示"积分充值"标签

2. **选择套餐**
   - 浏览可用的积分套餐
   - 查看积分数量和价格
   - 对比性价比

3. **点击充值**
   - 点击套餐卡片或"立即充值"按钮
   - 跳转到支付页面

4. **完成支付**
   - 选择支付币种（USDT/TRX）
   - 扫码或复制地址转账
   - 等待自动到账

### 开通VIP流程

1. **切换到VIP标签**
   - 点击顶部"VIP会员"标签

2. **选择套餐**
   - 浏览VIP套餐
   - 查看有效期和特权
   - 推荐套餐有特殊标识

3. **点击开通**
   - 点击套餐卡片或"立即开通"按钮
   - 跳转到支付页面

4. **完成支付**
   - 选择支付币种
   - 完成转账
   - VIP立即生效
