1. 登录管理员账号
2. 访问管理后台：`/admin`
3. 点击"充值系统配置"卡片
4. 或直接访问：`/admin/recharge-config`

在"BEpusdt支付配置"部分填写：

```

支持币种: ☑ USDT ☑ TRX
```

## 🔧 故障排除

### 问题1: 创建订单失败

**可能原因:**
- BEpusdt服务未运行
- API密钥错误
- 商户ID错误
- 网络连接问题

**解决方案:**
1. 检查BEpusdt服务状态
2. 确认配置信息正确
3. 查看浏览器控制台错误
4. 查看BEpusdt后台日志

### 问题2: 支付后未到账

**可能原因:**
- Webhook URL未配置
- 签名验证失败
- 后端未处理通知

**解决方案:**
1. 配置正确的Webhook URL
2. 检查签名密钥
3. 查看后端日志
4. 手动查询订单状态

### 问题3: 页面显示错误

**可能原因:**
- 配置数据格式错误
- localStorage数据损坏

**解决方案:**
1. 清除浏览器缓存
2. 重新保存配置
3. 检查浏览器控制台
